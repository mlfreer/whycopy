You start with endowment of 100 tokens, tokens are worthless on their own and you need to allocate the entire token budgets. Allocating one token to the option you buy one unit of the corresponding option. That is, on the example below 5 tokens are allocated to Option 2, that is the payment from this investment would be 350 if the coin comes up heads and 150 if the coin comes up tales. Given the total allocation the return would be 3700 if coin comes up heads and 5800 if coin comes up tales.

To change the allocation you can drag the horizontal bar or use the "+" and "-" buttons. You can also type directly the number of tokens you want to allocate to the Option in the input form.
